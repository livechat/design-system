### LoaderSpinner

The basic, atom component of Loader. You can use it in custom cases not handled by Loader.

#### Thin

```js
<div style={{ width: "56px", height: "56px" }}>
  <LoaderSpinner thickness="thin" />
</div>
```

```js noeditor
<ComponentHtmlMarkup>
  <div style={{ width: "56px", height: "56px" }}>
    <LoaderSpinner thickness="thin" />
  </div>
</ComponentHtmlMarkup>
```
