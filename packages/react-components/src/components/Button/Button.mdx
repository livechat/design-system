## Intro <a id="Intro" />

Button used to initialise user actions. Button labels clearly express the action that will occur when the user interacts with them.

<Canvas of={ButtonStories.Default} sourceState="none" />

#### Example implementation

```jsx
<Button kind="primary" size="large" onClick={() => {}}>
  Button text
</Button>
```

<BlockColumnsContainer>
  <BlockColumns label="When to Use">
    - Use buttons when you want users to perform an action or trigger a specific
    function. - Buttons are suitable for primary call-to-action (CTA) elements,
    secondary actions, or supporting functions.
  </BlockColumns>
  <BlockColumns isNegative label="When Not to Use">
    - Avoid using buttons when the action is purely informational and doesn't
    require user interaction. - Don't use buttons for non-actionable elements or
    decorative purposes.
  </BlockColumns>
</BlockColumnsContainer>
