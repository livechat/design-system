## Intro <a id="Intro" />

The Alert component is used to provide feedback to users about a particular action or situation.

<BlockColumnsContainer>
  <BlockColumns>
    It has three different sizes: - Small - Medium - Large
  </BlockColumns>
  <BlockColumns>
    And comes in four types: - Info - Error - Warning - Success
  </BlockColumns>
</BlockColumnsContainer>

#### Example implementation

```jsx
<Alert
  kind="info"
  primaryButton={{
    label: 'Primary button',
    handleClick: () => {},
  }}
  secondaryButton={{
    label: 'Link button',
    handleClick: () => {},
  }}
  onClose={() => {}}
>
  Alert message
</Alert>
```
