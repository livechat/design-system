## Available icons

These are the icons used in the LiveChat Design System. They are available as a separate package `@livechat/design-system-icons`.
To use them, import the icon you need and pass it to the `source` prop of the `Icon` component.

ℹ️ Click on the icon to copy its name to clipboard.
