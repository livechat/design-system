## Intro <a id="Intro" />

The List Item component is designed to be a universal styled element, suitable for use in various contexts such as menus,
navigation, or data presentation. It provides a structured way to display a collection of items, offering flexibility for customization.

This component can include individual items that may feature labels, icons, or additional context,
which can be positioned on either the left or right side of the items.

<Canvas of={ListItemStories.Default} sourceState="none" />

#### Example implementation

```jsx
<ListItem>Simple Option 1</ListItem>
// OR
<ListItem leftNode={<Icon icon="chat" />} rightNode={<Icon icon="chevron-down" />}>Complex option 2</ListItem>
```
