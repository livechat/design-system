## Intro <a id="Intro" />

`GlobalAlert` is a component that displays a global alert message. It is used to inform users about important information or changes in the system.
It is meant to be used in pair with the `AppFrame` component.

<Canvas of={Stories.Default} sourceState="none" />

#### Example implementation

```jsx
import { GlobalAlert } from '@livechat/design-system-react-components';

<GlobalAlert
  kind="info"
  label="Turn on notifications to never miss a chat!"
  onClose={handleSkip}
>
  <Button
    size="compact"
    kind="high-contrast"
    onClick={handleEnableNotifications}
  >
    Turn on browser notifications
  </Button>
</GlobalAlert>;
```
