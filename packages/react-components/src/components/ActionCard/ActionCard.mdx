## Intro <a id="Intro" />

`ActionCard` is a component used to create a tile layout. each card is clickable, triggering the given action.
Each component displays content in one column by default, but you can pass your own element via `secondColumn`, then the tile will turn into a two-column view.
By default, the component adapts to the width of the container in which it will be placed. If you want to keep it at a certain width, it will be best to use our own wrapper. Additionally, the component adjusts its layout depending on the width of the container in which it is located. Breakpoint is <b>623px</b>, if the container is smaller than this value, the component will change the layout from columns to rows.
