## Intro <a id="Intro" />

The Action Menu component presents a dropdown menu of related actions or options. It is typically
used to group related actions together and save space on the screen.

<Canvas of={ActionMenuStories.Default} sourceState="none" />

#### Example implementation

```jsx
<ActionMenu
  options={[
    {
      key: 'one',
      element: <ListItem>Option one</ListItem>,
      onClick: () => {},
    },
    {
      key: 'two',
      element: <ListItem>Option two</ListItem>,
      onClick: () => {},
    },
  ]}
  triggerRenderer={() => {}}
/>
```

<BlockColumnsContainer>
  <BlockColumns label="When to Use">
    - There are a set of related actions or options that the user may need to
    access quickly. - You want to reduce clutter on the screen by hiding
    secondary actions or options behind a dropdown menu. - You want to provide
    an alternative set of options to the user when a primary action is
    unavailable.
  </BlockColumns>
  <BlockColumns isNegative label="When Not to Use">
    - When there are only one or two actions that the user needs to perform. -
    When the actions or options are unrelated or not clearly related. - When the
    actions or options are essential to the user's task or workflow.
  </BlockColumns>
</BlockColumnsContainer>
