# ActionMenu

[Intro](#Intro) | [Content Spec](#ContentSpec) | [Component API](#ComponentAPI)

## Intro <a id="Intro" />

The Action Menu component presents a dropdown menu of related actions or options. It is typically 
used to group related actions together and save space on the screen.

<Canvas>
  <Story id="components-actionmenu--default" />
</Canvas>

#### Example implementation
```jsx
<ActionMenu
  options={[
    {key: 'one', element: (<ActionMenuItem>Option one</ActionMenuItem>), onClick: () => {}},
    {key: 'two', element: (<ActionMenuItem>Option two</ActionMenuItem>), onClick: () => {}},
  ]}
  triggerRenderer={() => {}}
/>
```

<BlockColumnsContainer>
  <BlockColumns label="When to Use">
    - There are a set of related actions or options that the user may need to access quickly.
    - You want to reduce clutter on the screen by hiding secondary actions or options behind a dropdown menu.
    - You want to provide an alternative set of options to the user when a primary action is unavailable.
  </BlockColumns>
  <BlockColumns isNegative label="When Not to Use">
    - When there are only one or two actions that the user needs to perform.
    - When the actions or options are unrelated or not clearly related.
    - When the actions or options are essential to the user's task or workflow.
  </BlockColumns>
</BlockColumnsContainer>
