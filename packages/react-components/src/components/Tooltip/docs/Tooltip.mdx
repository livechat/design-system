### Usage
#
<b>Usage</b>

- Used to clarify the meaning of an icon button 
- Short hint
- Contextual information
- Provide additional help or define an item or term

<b>Behavior on Hover</b>

Tooltips should appear on hover, focus on, click or tap on an element. Providing users with contextual information without the need for additional clicks.

<b>Content</b>

Text only

<b>Limitations</b>

Max <b>140 characters</b> to prevent overwhelming users with too much information.

<b>Size</b>

The size of the tooltip should be proportional to the length of the content. For short tooltips, a smaller size should be used, while longer tooltips should be larger to accommodate more content.
