# Avatar

[Intro](#Intro) | [Content Spec](#ContentSpec) | [Component API](#ComponentAPI)

## Intro <a id="Intro" />

The Avatar component is a visual representation of a user, typically displayed as a small circular image or 
icon. It comes in eight sizes and three types: Photo, Initials, and Fallback icon. This component helps to 
identify users and visually distinguish them in various contexts throughout an application.
