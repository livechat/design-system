# Colors

List of available color tokens. Click on token to copy it's value to clipboard.

## Usage

To use color tokens in your project, you need to import the `DesignToken` constant from `@livechat/design-system-react-components`:

```jsx
import { DesignToken } from '@livechat/design-system-react-components';
```

`DesignToken` contains all the color tokens available in the design system. You can use them in your project by using the `var()` function in CSS:

```css
.my-element {
  background-color: var(DesignToken.SurfaceBasicSubtle);
}
```

## Content

Content color tokens in a design system help organize information visually. By assigning specific colors to different elements, like using primary colors for main content and secondary colors for less important ones, designers create a clear hierarchy, improving user experience and overall design aesthetics.

## Surface

Surface colors can be used to create a visual hierarchy within a design, with different surfaces or layers of UI elements having different colors to indicate different levels of importance or information. For example, a primary surface color may be used for the main content area of a page, while secondary surface colors may be used for navigation bars or other less important elements.

### Primary

Primary surface, base for all content on the page. Primary surface color may be used for the main content area of a page, while secondary surface colors may be used for navigation bars, cards, banners or other less important elements.
