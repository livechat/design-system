# How to use?

Our Typography system could be used in two different ways:

1. css class
1. react component

## Css class

All classes in our Design System starts with `lc` prefix. Then, Typography tokens are build following the rule: `lc-TYPE-SIZE-MODIFIER`. Where `TYPE` can be `h` - for heading, `p` - for paragraph, `caps` for capitalized sub-heading. `SIZE` - from `xl` to `xs` for heading, and from `m` to `xs` for paragraph. To see supported modifiers please click on `Show Code` button below.

## React components

### Heading

Renders heading element with given size.

In `Heading` component you can modify the output element via `as` prop. For example if you want to render XLarge heading as `h3` just pass `as='h3'`

```jsx
<Heading size="xl" as="h3">
  My h3 heading
</Heading>
```

### Text

`Text` component renders a paragraph.

```jsx
<Text size="s" decoration="bold">
  This text is a small bold p
</Text>
```
